# Ansible Role: marvel-nccr.cp2k

An ansible role to install  a recent version of [CP2K](https://www.cp2k.org/) on Ubuntu.

## Installation

`ansible-galaxy install marvel-nccr.cp2k`

## Role Variables

See `defaults/main.yml`

## Example Playbook

  - hosts: servers
    roles:
    - role: marvel-nccr.cp2k
